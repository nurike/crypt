# crypt

Шифрование: encrypt.php -t Hello123456 -k 3
-t - Параметр для передачи текста для шифрования
-k - Ключ для шифрования. Ключ обязательно должен быть целочисленным.

Дешифрование: decrypt.php Khoor123456 -k 3
-t - Параметр для передачи текста для шифрования
-k - Ключ для шифрования. Ключ обязательно должен быть целочисленным.
